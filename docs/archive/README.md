# Archived docs (pre-consolidation)

These describe the **old** three-package architecture (`msp_pipeline`,
`cfo_pipeline`, `insurance_pipeline`) and the insurance niche, all of which
were removed when the pipelines were consolidated into the single `leadgen`
package.

They are kept only for historical reference. **They no longer reflect the
codebase.** For the current architecture see the root `README.md` and
`CLAUDE.md`.

- `M1.md`–`M10.md` — milestone plans for the old `msp_pipeline` build-out.
- `ROADMAP.md` — the old milestone roadmap.
- `INSURANCE_PLAN.md`, `INSURANCE_FIXES_BEFORE_REVIEW.md` — the removed
  insurance niche.
